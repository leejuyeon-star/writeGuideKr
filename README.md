## 주요 기능
### 회원 관리 기능
- 네이버, 구글 소셜 로그인&회원가입 기능
- 로그아웃 기능
- 탈퇴 기능
### ai 관련 기능
- 특정 문장구 이후에 나올 표현 추천 기능 
- 특정 단어를 드래그시 문맥에 어울리는 다른 단어 추천 기능
- ai 재요청 기능
- ai 요청시 1토큰 차감, 4시간 지나면 토큰 갱신
### 문서 편집 관련 기능
- undo, redo 기능
- 복사 기능
- 페이지 벗어나기 전 알림기능
- 글자수 세기 기능
- 커서 오른쪽 아래 ai 버튼 따라오도록 구현
